# movies-explorer-frontend

## Описание

2-я часть (frontend) дипломного проекта [Яндекс.Практикум](https://praktikum.yandex.ru/)

## Запуск проекта

`npm start` — запускает сервер   
`npm run build` — Создает оптимизированную версию приложения
